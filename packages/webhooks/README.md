# Papra webhooks

This package is used to trigger and easily consume webhooks from Papra.

## Installation

```bash
# using pnpm
pnpm add @papra/webhooks

# using npm
npm install @papra/webhooks

# using yarn
yarn add @papra/webhooks
```
